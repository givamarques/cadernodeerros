Resumo de Conceitos de Programação

1. Construção de Algoritmos

A construção de algoritmos é o processo de criação de sequências de passos lógicos para resolver problemas computacionais. Serve como base para a implementação de soluções em linguagens de programação.

2. Tipos de Dados Simples e Estruturados

Tipos de dados podem ser simples (como inteiros, caracteres) ou estruturados (como arrays e registros), e servem para definir como os dados são armazenados e manipulados em programas.

3. Variáveis e Constantes

Variáveis são locais na memória que armazenam valores que podem ser alterados durante a execução do programa. Constantes são valores fixos que não mudam durante a execução.

4. Comandos de Atribuição, Entrada e Saída

Os comandos de atribuição servem para definir ou modificar o valor de uma variável. Os comandos de entrada permitem a inserção de dados pelo usuário, enquanto os de saída exibem resultados.

5. Avaliação de Expressões

A avaliação de expressões consiste em calcular o resultado de expressões matemáticas ou lógicas, usadas para determinar o valor final que será armazenado ou exibido.

6. Funções Pré-definidas

Funções pré-definidas são funções fornecidas pela linguagem de programação para realizar tarefas comuns, como cálculos matemáticos (ex.: raiz quadrada) ou manipulação de strings.

7. Conceito de Bloco de Comandos

Um bloco de comandos é um conjunto de instruções agrupadas, geralmente delimitadas por chaves, que são executadas em sequência.

8. Estruturas de Controle: Seleção, Repetição e Desvio

Estruturas de controle determinam o fluxo de execução do programa. Seleção (if/else) toma decisões; repetição (loops) executa blocos várias vezes; e desvio (goto) altera a sequência de execução.

9. Operadores e Expressões

Operadores (matemáticos, lógicos, relacionais) são usados para criar expressões, que consistem em combinações de valores e operações que resultam em um valor.

10. Passagem de Parâmetros

A passagem de parâmetros é o processo de enviar valores ou referências para funções ou procedimentos, permitindo que eles operem sobre dados fornecidos externamente.

11. Recursividade

Recursividade é uma técnica onde uma função chama a si mesma para resolver problemas menores até alcançar um caso base. É usada em problemas que têm uma estrutura naturalmente repetitiva.

12. Conceitos Básicos de Programação Estruturada e Orientada a Objetos

A programação estruturada organiza o código em funções e procedimentos, enquanto a orientada a objetos organiza em classes e objetos, promovendo reuso e modularidade.

13. Métodos de Ordenação, Pesquisa e Hashing

Métodos de ordenação são usados para organizar elementos (ex.: Bubble Sort). Métodos de pesquisa localizam elementos específicos em estruturas de dados (ex.: busca binária). Hashing é uma técnica para mapear dados em uma estrutura de forma eficiente, facilitando operações de busca.
