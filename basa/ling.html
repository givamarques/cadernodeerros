LINGUAGENS E TECNOLOGIAS DE PROGRAMAÇÃO

Resumo de Tecnologias e Práticas de Desenvolvimento

1. Java

Java: Linguagem de programação orientada a objetos, amplamente utilizada para desenvolvimento de aplicações de diferentes portes, como web, desktop e mobile.

2. PHP

PHP: Linguagem de script do lado do servidor, utilizada principalmente no desenvolvimento de páginas web dinâmicas e integração com bancos de dados.

3. JavaScript

Linguagem de programação utilizada para adicionar funcionalidades interativas em páginas web. Funciona no lado do cliente, sendo essencial para aplicações web modernas.

4. Python

Linguagem de programação de alto nível conhecida por sua sintaxe simples e legibilidade. É utilizada em desenvolvimento web, automação, ciência de dados e inteligência artificial.

5. TypeScript

Superset do JavaScript que adiciona tipagem estática. Facilita a manutenção e a escalabilidade de grandes projetos JavaScript.

6. Ajax

Tecnologia que permite a atualização de partes de uma página web sem recarregá-la completamente, melhorando a experiência do usuário.

7. HTML5

Versão mais recente da linguagem de marcação usada para estruturar o conteúdo da web, com novos elementos semânticos e suporte aprimorado a multimídia.

8. CSS3

Última versão do CSS, utilizada para estilizar páginas web. Introduz novos recursos como animações e gradientes, permitindo layouts mais sofisticados.

9. DHTML

Dynamic HTML é uma combinação de HTML, CSS e JavaScript que permite criar páginas web dinâmicas e interativas.

10. XML, XSD, XSLT

XML: Linguagem de marcação para armazenar e transportar dados de forma estruturada.

XSD: Schema que define a estrutura e os tipos de dados em um documento XML.

XSLT: Linguagem para transformar documentos XML em outros formatos, como HTML.

11. JBoss Seam, JPA, Facelets, JNDI, JDBC

  • JBoss Seam: Framework para facilitar o desenvolvimento de aplicações Java EE, integrando vários componentes.
  • JPA (Java Persistence API): API usada para mapeamento objeto-relacional, simplificando o acesso a dados em Java.
  • Facelets: Tecnologia de visualização usada no JavaServer Faces (JSF) para criar interfaces web.
  • JNDI (Java Naming and Directory Interface): Interface para acessar diretórios e recursos em aplicações Java.
  • JDBC (Java Database Connectivity): API que permite a conexão de aplicações Java a bancos de dados.

12. Hibernate, Spring, JQuery

  • Hibernate: Framework ORM (Object-Relational Mapping) para facilitar a persistência de dados em aplicações Java.
  • Spring: Framework que simplifica o desenvolvimento de aplicações Java, fornecendo suporte a injeção de dependência e outras funcionalidades.
  • JQuery: Biblioteca JavaScript que facilita a manipulação do DOM e eventos, tornando o desenvolvimento web mais simples.

13. Práticas Ágeis

  • Integração Contínua: Técnica para integrar código frequentemente, garantindo a qualidade por meio de testes automatizados.
  • Controle de Versões e Repositórios: Sistemas como Git são usados para controlar mudanças no código e facilitar a colaboração entre desenvolvedores.
  • Test-Driven Development (TDD): Técnica de desenvolvimento em que os testes são escritos antes do código, garantindo maior qualidade e cobertura.
  • Refactoring: Processo de melhorar a estrutura do código sem alterar sua funcionalidade, tornando-o mais eficiente e fácil de manter.

14. Behaviour Driven Development (BDD)

Extensão do TDD que enfoca o comportamento do software do ponto de vista do usuário final. Utiliza linguagem natural para descrever funcionalidades.

15. Desenvolvimento para Plataforma Mobile

  • Android: Sistema operacional para dispositivos móveis desenvolvido pelo Google, utilizando Java/Kotlin para desenvolvimento de aplicativos.
  • iOS: Sistema operacional da Apple para dispositivos móveis, com aplicativos desenvolvidos em Swift.

16. React Native 0.59

Framework de desenvolvimento de aplicativos móveis multiplataforma, baseado em JavaScript, que permite criar aplicativos para Android e iOS utilizando uma única base de código.

17. Sistemas Android API 30 e iOS Xcode 10

Android API 30: Versão do Android SDK usada para desenvolver aplicativos com recursos modernos e compatibilidade com Android 11.

iOS Xcode 10: Ambiente de desenvolvimento integrado da Apple, utilizado para criar aplicativos para iOS com suporte ao Swift e Objective-C.
